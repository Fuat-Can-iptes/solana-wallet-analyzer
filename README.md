# 🚀 Solana Wallet Analyzer

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Solana](https://img.shields.io/badge/Solana-Network-green)

Masaüstü ortamında Solana ağındaki cüzdanları derinlemesine analiz etmek için geliştirilmiş, hızlı ve kullanıcı dostu bir uygulama. Bu araç; cüzdan bakiyelerini, işlem geçmişini ve ağ etkileşimlerini doğrudan masaüstünüzden takip etmenizi sağlar.

  ### Katkıda Bulunanlar
   *Bekir Şimşek
   *Fuat Can İpteş
   *Oğuz Şahan Saraçoğlu

## ✨ Özellikler

- **Masaüstü Arayüzü:** Hızlı ve akıcı kullanım için tasarlanmış grafik kullanıcı arayüzü (GUI).
- **Ağ Desteği:** Mainnet ve Devnet ortamları arasında kolay geçiş yaparak test ve analiz yapabilme.
- **Bakiye ve İşlem Sorgulama:** Anlık SOL bakiyesi ve geçmiş işlemleri (transaction history) listeleme.
- **API Optimizasyonu:** Yüksek istek oranlarında (rate-limiting) takılmaları önlemek için optimize edilmiş ağ isteği yönetimi.

## 🛠️ Kullanılan Teknolojiler

- **Platform:** Masaüstü Uygulaması (Desktop Application)
- **Arayüz (GUI):** Python (PyQt / CustomTkinter) vb.
- **Blockchain Entegrasyonu:** Solana RPC API
